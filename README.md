## Объяснение использования

### body
Настройка шрифта, цветов и тд.

### big
Большой и жирный текст

### text
Простой текст

### bold
Жирный текст

### block
Квадрат

### content
Отступ от края

### end
Текст серого цвета, который подходит для подвала страницы

### @media
Определение темы ОС
